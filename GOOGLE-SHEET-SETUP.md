# ຕັ້ງຄ່າ Google Sheet (ໃຊ້ຂໍ້ມູນຮ່ວມກັນຫຼາຍຄົນ)

ເຮັດຕາມ 4 ຂັ້ນຕອນນີ້ ເທື່ອດຽວ. ຫຼັງຈາກນັ້ນທຸກຄົນທີ່ເປີດເວັບ ຈະເຫັນຂໍ້ມູນອັນດຽວກັນ ແລະ ໃຜແກ້ກໍ່ອັບເດດໃຫ້ທຸກຄົນ.

---

## ຂັ້ນຕອນ 1 — ສ້າງ Google Sheet
1. ໄປທີ່ https://sheets.new (ສ້າງ Sheet ໃໝ່)
2. ຕັ້ງຊື່ຕາມໃຈ ເຊັ່ນ **Countdown Data**
3. ບໍ່ຕ້ອງໃສ່ຫຍັງໃນ Sheet — ໂປຣແກຣມຈະສ້າງແຖວຫົວຕາຕະລາງ (sheet ຊື່ `Data`) ໃຫ້ເອງ

## ຂັ້ນຕອນ 2 — ໃສ່ໂຄ້ດ Apps Script
1. ໃນ Google Sheet ນັ້ນ → ເມນູ **Extensions ▸ Apps Script**
2. ລຶບໂຄ້ດເກົ່າທີ່ມີຢູ່ໃຫ້ໝົດ
3. ເປີດໄຟລ໌ [`google-apps-script/Code.gs`](google-apps-script/Code.gs) → **ກັອບປີ້ທັງໝົດ** ໄປໃສ່
4. ກົດ 💾 (Save)

## ຂັ້ນຕອນ 3 — Deploy ເປັນ Web App
1. ກົດ **Deploy ▸ New deployment**
2. ໄອຄອນ ⚙️ (Select type) → ເລືອກ **Web app**
3. ຕັ້ງຄ່າ:
   - **Description**: ຫຍັງກໍ່ໄດ້ (ເຊັ່ນ countdown api)
   - **Execute as**: **Me** (ຕົວທ່ານເອງ)
   - **Who has access**: **Anyone**  ← ສຳຄັນ! ຕ້ອງເລືອກ "Anyone"
4. ກົດ **Deploy**
5. ຄັ້ງທຳອິດ Google ຈະຂໍ **Authorize** → ກົດ Authorize, ເລືອກບັນຊີ, ກົດ **Advanced ▸ Go to … (unsafe)** → **Allow**
   *(ມັນບອກວ່າ "unsafe" ເພາະເປັນສະຄຣິບຂອງເຮົາເອງ — ປອດໄພ)*
6. ຈະໄດ້ **Web app URL** ໜ້າຕາແບບນີ້:
   ```
   https://script.google.com/macros/s/AKfy............/exec
   ```
   **ກັອບປີ້ URL ນີ້ໄວ້**

## ຂັ້ນຕອນ 4 — ໃສ່ URL ໃນເວັບ
1. ເປີດໄຟລ໌ [`index.html`](index.html) ດ້ວຍ Notepad / VS Code
2. ຫາແຖວ:
   ```js
   var SHEET_API_URL = '';
   ```
3. ໃສ່ URL ທີ່ກັອບປີ້ມາ ໃສ່ໃນເຄື່ອງໝາຍຄຳ:
   ```js
   var SHEET_API_URL = 'https://script.google.com/macros/s/AKfy............/exec';
   ```
4. Save

ສຳເລັດ ✅ — ເປີດ `index.html` ໃໝ່ ຈະເຫັນປ້າຍ **"ຊິງຄ໌ Google Sheet ✓"** ຢູ່ມຸມຂວາເທິງ.

---

## ໃຫ້ຄົນອື່ນໃຊ້ນຳ
ໄຟລ໌ເວັບ (`index.html` + ໂຟນເດີ `images/`) ຕ້ອງຢູ່ບ່ອນທີ່ທຸກຄົນເຂົ້າເຖິງໄດ້. ເລືອກວິທີໃດໜຶ່ງ:

| ວິທີ | ເໝາະກັບ |
|---|---|
| **ສົ່ງໄຟລ໌ໃຫ້ກັນ** (USB / chat) — ແຕ່ລະຄົນເປີດໃນເຄື່ອງຕົນ | ທີມນ້ອຍ, ງ່າຍສຸດ |
| **GitHub Pages / Netlify / Vercel** (ຟຣີ) — ໄດ້ລິ້ງເວັບ ສົ່ງໃຫ້ທຸກຄົນ | ໃຫ້ໃຊ້ໃນມືຖື/ຄອມໃດກໍ່ໄດ້ |
| **Google Drive** (ແບ່ງປັນໂຟນເດີ) | ມີ Drive ຢູ່ແລ້ວ |

> ທຸກຄົນໃຊ້ **URL ດຽວກັນ** ໃນ `SHEET_API_URL` → ເຫັນຂໍ້ມູນດຽວກັນ.
> ກົດປຸ່ມ **↻ ໂຫຼດໃໝ່** ມຸມຂວາ ເພື່ອດຶງຂໍ້ມູນຫຼ້າສຸດທີ່ຄົນອື່ນແກ້ໄວ້ (ກໍ່ໂຫຼດເອງຕອນເປີດ/ສະຫຼັບກັບມາໜ້າຕ່າງ).

## ສິ່ງທີ່ຄວນຮູ້
- **ຮູບສິນຄ້າທີ່ລາກປ່ຽນເອງ** ບໍ່ sync ຜ່ານ Sheet (ໄຟລ໌ໃຫຍ່ເກີນ) — ເກັບແຍກໃນແຕ່ລະເຄື່ອງ. ຮູບເລີ່ມຕົ້ນ 4 ຮູບ ທຸກຄົນເຫັນຄືກັນ.
- ຂໍ້ມູນທີ່ sync = ຊື່ສິນຄ້າ, ງວດ, ວັນທີສັ່ງ/ຄາດການ, ຍອດຍັງບໍ່ເກັບເງີນ, ງວດທີ່ກຳລັງສະແດງ.
- ເປັນແບບ **last-write-wins**: ຖ້າ 2 ຄົນກົດ "ບັນທຶກ" ພ້ອມກັນ, ອັນທີ່ບັນທຶກທີຫຼັງຈະທັບ. ສຳລັບທີມນ້ອຍບໍ່ມີບັນຫາ.
- ຖ້າແກ້ໂຄ້ດ `Code.gs` ພາຍຫຼັງ → ຕ້ອງ **Deploy ▸ Manage deployments ▸ ✏️ Edit ▸ Version: New version ▸ Deploy** (URL ຄືເກົ່າ).
- ຢາກເບິ່ງ/ແກ້ຂໍ້ມູນກົງໆ ກໍ່ເປີດ Google Sheet ໄດ້ເລີຍ (sheet ຊື່ `Data`).
